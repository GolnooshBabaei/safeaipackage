# Safeai Crew

Welcome to the Safeai

## Installation


## Clone SafeAI Github Repo

```bash
pip install poetry
git clone https://github.com/GolnooshBabaei/safeaipackage.git
cd safeai
```

## Create Virtual Environment

```bash
python -m venv .venv
source .venv/bin/activate
```

## Install Dependencies
```bash
poetry install
```

## Kickoff Sample Experiment
```bash
python3 sample.py
```
